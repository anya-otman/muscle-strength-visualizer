# Динамика мышечной силы

Локальное веб-приложение на Python (Streamlit) для визуализации динамики мышечной силы пациента по данным из Excel (`.xlsx`).

Проект предназначен для анализа изменений мышечной силы во времени, с наглядным представлением:
- 2D-графиков по каждому движению
- 3D-модели ноги (бедро, голень, стопа), где цвет сегмента отражает среднюю мышечную силу

---

## Возможности

- Загрузка Excel-файла `.xlsx`
- Выбор пациента (ФИО в формате `Пациент N`, уникальный идентификатор)
- 2D-графики динамики мышечной силы:
  - каждый график — отдельное движение
  - ось X — дата осмотра
  - ось Y — мышечная сила (0–5)
- 3D-визуализация ноги:
  - сегменты: бедро / голень / стопа
  - цвет сегмента = средняя мышечная сила по группе на выбранную дату
- Игнорирование столбцов с суффиксом `до` (показатели до лечения)

---

## Требования

- Python **3.9+** (рекомендуется 3.10 или 3.11)

---

## Установка

### 1. Клонирование репозитория

```bash
git clone git@github.com:anya-otman/muscle-strength-visualizer.git
cd muscle-strength-visualizer
```

### 2. Установка зависимостей
```
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

### 3. Запуск приложения
```
python -m streamlit run app.py
```

## Формат входного Excel-файла

Файл должен содержать следующие обязательные колонки:

- Дата осмотра

- ФИО

- Возраст

Также в файле должны быть колонки с показателями мышечной силы (значения от 0 до 5).

## Правила обработки данных

Колонки, оканчивающиеся на до, автоматически игнорируются

Для одного пациента — одна запись в один день

Пустые значения и текст вместо чисел не допускаются (валидируются при загрузке)

Шкала мышечной силы: 0–5 (MRC scale)

## 3D-визуализация

3D-модель ноги является схематичной

Используется для наглядного отображения динамики по анатомическим сегментам
